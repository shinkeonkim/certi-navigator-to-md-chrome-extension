# Certi Navigator to Markdown Chrome Extension

이 크롬 익스텐션(Chrome Extension)은 Certi Navigator 등의 퀴즈/문제 풀이 웹페이지에서 문제 내용, 보기, 정답, 해설을 추출하여 마크다운(`.md`) 파일로 저장하거나, 클립보드에 복사하거나, 사용자가 지정한 디렉토리에 바로 저장할 수 있게 해주는 도구입니다.

- **Github Repository**: [https://github.com/shinkeonkim/certi-navigator-to-md-chrome-extension](https://github.com/shinkeonkim/certi-navigator-to-md-chrome-extension)

## 기능 (Features)
- 현재 보고 있는 웹페이지의 문제 제목 추출
- 단일/다중 선택형 보기 옵션 추출 (체크박스 형태로 포맷팅)
- 정답 및 해설이 있는 경우 추가로 추출
- 일관된 Markdown 문서 형태로 자동 다운로드 (문제 제목을 파일명으로 사용)
- 추출된 마크다운 텍스트를 **클립보드에 바로 복사**
- 사용자가 지정한 **특정 디렉토리에 바로 저장** (File System Access API 사용)
  - 기본값 없음. 처음 사용 시 한 번만 폴더를 선택하면 이후 자동 저장
  - 동일한 파일명이 이미 존재하면 자동으로 타임스탬프 suffix 추가 (e.g., `제목-20260426-153012.md`)
- **파일명 직접 지정** (선택사항): popup의 입력 필드에서 원하는 파일명을 입력하면 다운로드/폴더 저장 시 해당 이름으로 저장. 비워두면 기존처럼 문제 제목으로 자동 생성. `.md` 확장자는 자동 처리됨.

## 설치 방법 (Installation)
1. 해당 [GitHub Repository](https://github.com/shinkeonkim/certi-navigator-to-md-chrome-extension)를 클론하거나 폴더를 다운로드합니다.
2. 크롬 브라우저에서 `chrome://extensions/` 주소로 접속합니다.
3. 우측 상단의 **'개발자 모드(Developer mode)'**를 활성화합니다.
4. **'압축해제된 확장 프로그램을 로드합니다(Load unpacked)'** 버튼을 클릭하고 다운로드한 폴더를 선택합니다.

## 사용 방법 (Usage)
1. 문제가 있는 웹페이지로 이동합니다.
2. 브라우저 우측 상단의 크롬 브라우저 확장 프로그램 아이콘 리스트에서 이 익스텐션의 아이콘을 클릭하여 팝업을 엽니다.
3. (선택) **파일명 입력 필드**에 원하는 파일명을 입력합니다. 비워두면 문제 제목으로 자동 생성되며, `.md` 확장자는 자동 추가됩니다. (클립보드 복사에는 적용되지 않음)
4. 원하는 동작에 맞는 버튼을 클릭합니다.
   - **'Markdown으로 다운로드'**: 변환된 `.md` 파일이 브라우저의 기본 다운로드 폴더에 저장됩니다.
   - **'클립보드에 복사'**: 변환된 마크다운 텍스트가 시스템 클립보드에 복사되어, 다른 에디터나 노트 앱에 바로 붙여넣을 수 있습니다.
   - **'폴더에 저장'**: 미리 설정한 디렉토리에 마크다운 파일이 직접 저장됩니다.

### 저장 폴더 설정 (Directory Setup)
- 팝업 하단 **'저장 폴더'** 영역에서 **'폴더 선택'** 버튼을 클릭하여 원하는 폴더를 선택합니다.
- 한 번 설정하면 브라우저에 영속 저장되어 다음에 다시 선택할 필요가 없습니다.
- 폴더를 변경하려면 **'폴더 변경'** 버튼을, 설정을 해제하려면 **'해제'** 버튼을 클릭합니다.
- 폴더가 설정되지 않은 상태에서 **'폴더에 저장'** 버튼을 클릭하면 자동으로 폴더 선택 다이얼로그가 표시됩니다.

> **Note**: 브라우저가 폴더 쓰기 권한을 일정 기간 후 만료시킬 수 있으며, 이 경우 다시 권한을 요청합니다. 폴더가 삭제되거나 이동되면 권한이 무효화될 수 있고, 그 때는 폴더를 다시 선택해야 합니다.
